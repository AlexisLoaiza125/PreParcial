Pokémon Battle API - FastAPI
Pequeña API hecha con FastAPI que permite consultar Pokémon y simular batallas entre ellos.

1. Crear un entorno virtual (venv)
Es recomendable usar un entorno virtual para aislar las dependencias del proyecto.

2. Instalar dependencias
Instala las librerías necesarias del proyecto usando el archivo requirements.txt.
pip install -r requirements.txt

4. Funcionalidades de la API

La API permite:
  Obtener todos los Pokémon
  Buscar Pokémon por nombre
  Buscar Pokémon por id
  Ordenar Pokémon por vida
  Simular batallas entre Pokémon
